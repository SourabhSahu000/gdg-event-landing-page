# GDG Event Landing Page

A responsive landing page for a fictional Google Developer Group (GDG) event.

## Features
- Event details (title, date, venue)
- Event description section
- Speakers section with cards
- Functional Register button
- Countdown timer to event date
- Dark mode toggle
- Responsive design for mobile, tablet, and desktop
- Clean and modern UI with cherry blossom themed background

## Tech Stack
- HTML
- CSS
- JavaScript

## Live Demo
https://sourabhsahu000.github.io/gdg-event-landing-page/

## Purpose
This project was built as part of a frontend assessment challenge to demonstrate basic web development fundamentals, UI/UX sense, and clean layout design.
